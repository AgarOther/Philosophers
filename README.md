# Description
Philosophers is a project that aims to sharpen your knowledge on multithreading and mutexes. It teaches how to manage a multithreaded environment in order to avoid data races when multiple threads try accessing the same values.

# Instructions
Compile the project using
```
make
```

Then start the executable using
```
./philo <number of philosophers> <time to die> <time to eat> <time to sleep> (amount of meals)
```

# Resources

No AI was used in this project, I used the man for the pthread.h functions and a few stackoverflow issues when I was stuck. Also got help from a few students at 42 for lock orders.
